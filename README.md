## How to Run Locally

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) (v14+ recommended)

### Steps to Run

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm start
   ```

3. Open your browser and go to:

   ```
   http://localhost:3000
   ```
